Partclone

Partclone provides utilities to smartly backup partitions. Partclone is written in C and designed for higher compatibility by using existing file system library.

Clonezilla

You can use clonezilla which is an integrated tool to back your disk/partition. Clonezilla, based on DRBL, Partition Image, ntfsclone, partclone, and udpcast, allows you to do bare metal backup and recovery. go to clonezilla
